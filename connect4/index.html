Connect 4: Playable AI opponents!

This project is a fully playable, end-to-end AI implementation of Connect 4, designed to explore how different modeling approaches translate into real gameplay. The system was trained on a large, strategically diverse dataset generated using Monte Carlo Tree Search, allowing the AI to learn optimal move selection across a wide range of board states. Two distinct models power the game: a convolutional neural network optimized for fast, consistent play, and a transformer-based model designed to reason over broader board patterns. Both models are deployed on a cloud backend and integrated into a live web interface, enabling real-time play and comparison. Beyond model accuracy, the project emphasizes product trade-offs—responsiveness, difficulty, and user experience—demonstrating how AI design decisions surface directly in an interactive game environment.
